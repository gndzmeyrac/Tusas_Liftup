# TUSAŞ LiftUp

React (Vite) ile gelistirilmis 360° degerlendirme ve gelisim raporu arayuzu.

## Gereksinimler

- Node.js 18+
- npm

## Kurulum

```bash
npm install
```

## Calistirma

```bash
npm run dev
```

## Uretim Derlemesi

```bash
npm run build
```

## Yardimci Scriptler

```bash
npm run sync:egitimler
npm run complete:faz0
npm run merge:tavsiye
```
